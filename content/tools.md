I often get asked about what tools I use for Shopify development and how I manage both freelance work and university side by side. Here's everything I use daily built around speed, clarity, and getting actual work done.
My motivation is simple: use tools that don't slow me down. Whether I'm customizing a theme, writing Liquid logic, or working through a cost accounting problem, I want my tools out of the way

![setup image](/assets/desktop.webp)

## Shopify Development

- MY primary code editor is **[Vs Code](https://code.visualstudio.com/)** With extensions like Shopify Liquid, Prettier, and GitLens, it becomes a complete Shopify development environment for Liquid, CSS, and JavaScript work.

- **[Shopify Partner](https://http://partners.shopify.com//)** The official hub for **managing client stores, development stores, app builds, and theme submissions.** Essential for any professional Shopify developer handling multiple client accounts and projects.

- **[Figma](https://www.figma.com/)** The industry-standard design tool for UI/UX collaboration.I Used to review client design specs, convert mockups to Liquid theme code, and prototype Shopify storefront layouts before development.

- **[Postman](https://www.postman.com/)** API testing and debugging tool for working with Shopify's Admin REST API and GraphQL API. Essential for building custom apps, third-party integrations, and automating store data operations.

- **[Google Search Console + Analytics](https://search.google.com/search-console/about)** Core tools for Shopify store SEO and performance tracking. Search Console monitors indexing and keyword rankings; Analytics tracks user behavior, conversion funnels, and traffic sources to optimize store revenue.

- For browser extensions, I keep things minimal:

    - uBlock Origin to block ads and trackers.
    - Koala Inspector to analyse the store
    - ColorZilla to get a color reading from any point in your browser, quickly adjust this color and paste it into another program.
    - whatfont to identify font on web pages.

## Finance Tools

- **[Microsoft Excel](https://excel.cloud.microsoft/)** The backbone of financial work. Used for financial modeling, budgeting, cost analysis, variance reporting, and building management accounting schedules. Pivot tables and advanced formulas are essential for CMA-level work.

- **[Xero](https://www.xero.com/)** Cloud-based accounting platforms for bookkeeping, accounts payable/receivable, payroll, and financial statement generation. Xero is widely adopted for its clean interface and real-time financial data sync.

- **[Power BI](https://app.powerbi.com/)** Microsoft's business intelligence suite for building interactive financial dashboards, KPI tracking, and management reporting. Integrates directly with Excel and accounting platforms for seamless data visualization.

- **[Zoho Books](https://www.zoho.com/books/)** A cost-effective accounting solution suited for small to mid-sized businesses. Handles invoicing, tax compliance, bank reconciliation, and financial reporting with good Shopify integration as a bonus.

## Applications  

I use **notion** for managing client projects, tracking my study progress, creating SOPs, and maintaining a personal knowledge base. Acts as a command center across both professional disciplines.  

**Gmail + Google Workspace** Professional email and document suite for client communication, sharing reports, collaborating on spreadsheets, and maintaining a polished professional presence across accounting and development clients.  

For Team Communication I use **slack**: real-time client updates, coordinating with design teams, and staying plugged into the Shopify developer community and partner networks.

For note-taking, I use **Obsidian**. I haven’t been using it for very long, and I’m still exploring its features, but so far, it’s been great for organizing knowledge effi  

For reading books, I use **Foliate**, which provides a clean and distraction-free experience.  

For Telegram, I prefer **64gram**, as it’s lightweight and efficient.  

For office work, I use **microsoft 365**   

For an application launcher, I use **Tofi**. I previously used Rofi, but I had some issues with it on Wayland. Tofi is simpler, highly customizable, and works flawlessly. I also use it as a power menu.
